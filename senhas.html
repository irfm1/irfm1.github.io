Senhas Seguras

Senhas são a primeira linha de defesa para proteger suas contas e dados pessoais. Utilizar senhas fortes e únicas para cada serviço é fundamental para evitar invasões e golpes.

Infográfico: Como Criar uma Senha Forte
📏

Comprimento é Poder!

Mínimo de 12 caracteres, idealmente 16 ou mais. Senhas longas são exponencialmente mais difíceis de quebrar.

🔀

Misture e Combine!

Use uma combinação de:

  • Letras maiúsculas (A-Z)
  • Letras minúsculas (a-z)
  • Números (0-9)
  • Símbolos (!@#$%^&*)
🔒

Seja Único, Sempre!

Nunca reuse senhas. Cada conta deve ter uma senha exclusiva. Se uma conta for comprometida, as outras permanecerão seguras.

🚫

Evite o Óbvio!

Não use informações pessoais facilmente descobertas:

  • Datas de nascimento
  • Nomes de familiares ou pets
  • Sequências como "123456" ou "abcdef"
  • Palavras do dicionário
💬

Pense em Frases!

Use frases secretas longas e memoráveis. Exemplo:
"Meu café favorito custa R$5,50 na padaria!"
Transforme em: McfcR$5,50np!

🔑

Use um Gerenciador de Senhas!

Ferramentas como 1Password, Bitwarden ou LastPass podem:

  • Gerar senhas complexas automaticamente
  • Armazenar senhas de forma segura
  • Preencher formulários automaticamente
  • Sincronizar entre dispositivos
📲

Ative a Autenticação de Dois Fatores (2FA)!

Adicione uma camada extra de segurança:

  • SMS (menos seguro, mas melhor que nada)
  • Apps autenticadores (Google Authenticator, Authy)
  • Chaves de segurança físicas (mais seguro)
⚠️

Sinais de Alerta!

Troque suas senhas imediatamente se:

  • Receber alertas de login suspeito
  • Notar atividade estranha nas contas
  • Uma empresa reportar vazamento de dados
  • Compartilhou a senha acidentalmente
